## App Icon Sizes (macOS Standards)
All generated from the clean cropped source:

- **icon_1024x1024.png** - App Store and high-resolution displays
- **icon_512x512.png** - Retina displays @2x for 256pt
- **icon_256x256.png** - Standard displays @1x for 256pt
- **icon_128x128.png** - Retina displays @2x for 64pt, standard @1x for 128pt
- **icon_64x64.png** - Retina displays @2x for 32pt
- **icon_32x32.png** - Retina displays @2x for 16pt, standard @1x for 32pt

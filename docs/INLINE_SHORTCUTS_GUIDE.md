## üö® Troubleshooting

### Shortcut Not Working?
1. Check Console.app for "AudioSwitchPro" messages
2. Ensure app has Accessibility permissions
3. Try a different key combination
4. Restart the app if needed

### Can't Record Shortcut?
1. Make sure you clicked "Set Shortcut" first
2. Include at least one modifier key
3. Press ESC to cancel and try again

### Permission Issues?
- Go to System Preferences > Security & Privacy > Privacy > Accessibility
- Ensure AudioSwitch Pro is checked
